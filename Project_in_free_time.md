# # 10th Grade Marks & Grade Calculator
A Python-based utility to calculate subject-wise grades, total marks, and overall percentages for 10th-grade students.

## 📌 Project Overview
This project was developed as a logic-building exercise during my **180-Day Data Science Challenge**. It focuses on mastering **conditional branching** and **nested logic** to handle real-world grading criteria.

## 🛠️ Features
- **Subject-Specific Grading:** Calculates grades (A+ to D) for Telugu, English, Maths, Physics, and Chemistry.
- **Nested Logic:** Verifies if a student has passed (>=35) before assigning a grade.
- **Error Handling:** Includes edge-case logic for perfect scores (100/100).
- **Automated Results:** Provides total marks and percentage calculation.

## 💻 Logic Used
- `if-else` statements for pass/fail verification.
- `nested if` for granular grade classification.
- Arithmetic operators for percentage and summation.

## 🚀 How to Run
1. Ensure you have Python installed.
2. Run the script: `python "Marks calculator.py"`
3. Enter marks for the five subjects when prompted.
